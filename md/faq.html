Frequently asked questions

What happened to the random section?

The random section required a lot of resources to run due to how terribly it was coded. If you really need an alternative, I recommend Randeeb by Howl

I don't have videos in WebM or MP4 format. Why don't you accept other video types?

I don't accept other file types as they're not optimized for the internet. (In fact, not all MP4 files are either) If you want to convert videos to WebM (Recommended format) for uploading, download the newest release of 4chan's WebM converter and run your file through that.

Do you look at all my files?

I do not take a detailed look on files. However, I skim across image thumbnails as I am required to due to Norway's laws (To check for child porn and all that stuff) I will however never share images found while moderating.

How much does your server cost?

Roughly $10 per month. It's quite an affordable VPS and for now it seems to be enough. In the future I might have to move to a dedicated server though, and that can get quite expensive.

QuadFile is slow, what's going on?

It's probably under heavy load. As the server is only a VPS, the network connection is shared. Due to this I depend on CloudFlare's CDN service to lower the bandwidth load on the server, but at times, it's simply not enough.
After a while it'll probably get back on its feet with proper speed.

How much storage does the server have?

This is scalable, so the answer is: However much I might need.

What would happen if QuadFile grew past what you can afford to maintain?

There are 3 possible options, that I would even consider:

  • Adding ads to make the service self-sustaining.
  • Placing the server in Read-only mode so that no new files could be uploaded, but existing files would still be accessible.
  • QuadFile gets shut down (Pretty unlikely)

Other than this I might ask for donations when problems start to occur, but for now it's running nicely though.
